--------------------------------------------------------------------------------
PROGRESSION RECAP: PHASES 1–3
--------------------------------------------------------------------------------

Phase 1 (Baseline)
  - 7 models × 3 input configurations, blind PCA (701 → 11 components)
  - Config B (discharge params only): Ridge/PLS R² ≈ 0.90 — best result
  - Config A (raw OES): R² ≈ 0.50–0.70 — severely hurt by dimensionality
  - Config C (OES + discharge, PCA): R² < 0.50 for most models
  - Key finding: B >> C > A; OES alone fails under blind PCA

Phase 2 (Bayesian Hyperparameter Tuning)
  - Applied Optuna to RF, MLP, CNN
  - Tuning improved all models universally (MLP Config C: −1.13 → 0.37; CNN Config C: 0.69 → 0.77)
  - B >> C > A pattern persisted — tuning alone could not overcome the PCA bottleneck
  - Best Config C result: CNN R² = 0.77

Phase 3 (Domain-Knowledge Feature Engineering)
  - Replaced blind PCA with 13 hand-crafted OES features (7 single-wavelength intensities,
    3 band integrals, 3 intensity ratios) + 4 discharge parameters = 17 features (Config C)
  - Ridge Config C: R² = 0.80; MLP Config C: R² = 0.81 — both surpassed Phase 2 best CNN
  - Key finding: domain expertise > deep learning on small datasets (n = 20)

--------------------------------------------------------------------------------
PHASE 4 GOALS
--------------------------------------------------------------------------------

Phase 4 shifted from "how well can we predict?" to:
  - WHY do these predictions work? (interpretability)
  - HOW RELIABLE are our conclusions? (statistical stability, n = 20)
  - CAN we simplify further? (feature redundancy and pruning)

9. PHYSICAL INTERPRETATION
   -------------------------
   Most consistently important OES signals (across models):
   - band_CO2p_398_412 (CO₂⁺ Fox-Duffendack-Barker band, rank #2):
     Tracks CO₂ ionisation → atomic O production → OH generation via O + H₂O → 2·OH.
     CO₂ dissociation is identified as a key rate-limiting step.
   - I_486_Hb (Hβ Balmer line, rank #4):
     Traces H atom population from H₂O dissociation, directly feeding OH radical pool.
   - band_CO_Hb_460_500 (CO + Hβ composite, rank #4):
     Captures both CO₂ decomposition products and H₂O dissociation.

   Counterintuitive finding: I_309_OH (direct OH emission, the immediate H₂O₂
   precursor) ranks only #16 in consensus. Its signal is redundant — captured
   more reliably by band_OH_306_312 and ratio_309_656. Models correctly learn
   to use denoised proxies rather than the noisy raw single-wavelength measurement.

--------------------------------------------------------------------------------
CORE CONCLUSIONS
--------------------------------------------------------------------------------

1. Feature–target relationships are predominantly LINEAR.
   Ridge and MLP Config C produce statistically indistinguishable results
   (Bootstrap CIs overlap fully). Non-linear models add no significant gain.
   For small OES datasets, Ridge is preferred: simpler, interpretable,
   deterministic, and equally accurate.

2. Discharge parameters dominate prediction (for linear models).
   Ridge assigns 69% of importance to 4 discharge parameters. This directly
   explains why Config B (discharge only) consistently rivals or beats Config C
   (OES + discharge) when all 13 OES features are included without pruning.

3. The 13 OES features are severely redundant — and pruning improves performance.
   9/13 features have VIF > 10. Removing redundant OES features monotonically
   improves Ridge Config C from R² = 0.798 → 0.918 (+0.120). The final pruned
   model (7 features) outperforms all Phase 1–3 configurations.

4. Ratios and band integrals outperform single-wavelength intensities.
   3 ratios (R² = 0.906) > 7 single-wavelength intensities (R² = 0.823).
   Self-normalised ratios compensate for total emission fluctuations; band
   integrals smooth pixel-level noise. Both categories effectively denoise OES.

5. MLP interpretability is unreliable with n = 20.
   82% of MLP SHAP values are unstable (CV > 1.0) across LOOCV folds.
   Ridge coefficients are stable (6% instability). On small datasets, linear
   models provide more trustworthy feature attributions than neural networks.

6. CO₂⁺ and Hβ — not direct OH — are the key OES diagnostic markers.
   Monitoring band_CO2p_398_412 and I_486_Hb provides a more reliable real-time
   proxy for H₂O₂ yield than the direct OH 309 nm line. This suggests a new
   practical strategy: use upstream plasma chemistry markers (CO₂ ionisation,
   H₂O dissociation) rather than the end-product OH radical for online monitoring.
